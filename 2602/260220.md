## 비트 연산자 정리

| 연산자 | 이름 | 연산 규칙 | 주요 용도 |
| :---: | :--- | :--- | :--- |
| **`&`** | **AND** | 둘 다 1일 때만 1 | 특정 위치에 1이 있는지 확인 (Masking) |
| **`\|`** | **OR** | 하나라도 1이면 1 | 두 상태를 합치거나 특정 위치에 1 추가 |
| **`^`** | **XOR** | 서로 다를 때만 1 | 두 값이 같은지 비교, 중복 숫자 제거 |
| **`~`** | **NOT** | 0은 1로, 1은 0으로 | 비트 반전 (보수 계산) |
| **`<<`** | **L-Shift** | 비트를 왼쪽으로 이동 | 숫자에 2의 거듭제곱을 곱함 |
| **`>>`** | **R-Shift** | 비트를 오른쪽으로 이동 | 숫자를 2의 거듭제곱으로 나눔 |

#### 비트 연산 사용
1. **짝수/홀수 확인** : `if ((N & 1) == 0)`
2. **2의 거듭제곱 확인** : `if ((N & (N - 1)) == 0)` 
3. **n번째 비트 켜기** : `number |= (1 << n)`
4. **n번째 비트 끄기** : `number &= ~(1 << n)`
