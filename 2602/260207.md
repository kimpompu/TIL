# 260207 TIL

## 메소드
| 메소드 | 설명 | 사용 예 |
| :--- | :--- | :--- |
| `String.replace(target, replacement)` | 문자열 내의 특정 문자열을 찾아 다른 문자열로 모두 바꿈 | `"one2one".replace("one", "1")` -> `"121"` |
| `String.valueOf(value)` | 다른 타입의 값을 문자열로 변환 | `String.valueOf(7)` -> `"7"` |
| `Long.parseLong(string)` | 숫자 형태의 문자열을 `long`으로 변환 | `Long.parseLong("123")` -> `123L` |
| `sb.charAt(index)` | 특정 인덱스에 있는 문자 반환 | `sb.charAt(1)` -> `'e'` |
| `sb.setCharAt(index, char)` | `StringBuilder` 내의 특정 인덱스 문자를 변경 | `sb.setCharAt(0, 'z')` |
| `IntStream.rangeClosed(s, e)` | `s`부터 `e`까지 포함하는 정수 범위를 생성 | `rangeClosed(1, 3)` -> `1, 2, 3` |
| `.filter(predicate)` | 조건에 맞는 데이터만 필터링 | `.filter(i -> i % 2 == 0)` |
