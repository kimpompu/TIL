## 최대공약수(GCD), 최소공배수(LCM)
```Java
public int gcd(int a, int b) {
    return b == 0 ? a : gcd(b, a % b);
}

public int lcm(int a, int b) {
    return a * b / gcd(a, b);
}
```

long result = 1;
