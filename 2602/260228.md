## Arrays.binarySearch(arr, key)
###반환
- `양수` 또는 `0` : 찾고자 하는 값이 배열에 존재하는 경우, 해당 요소의 인덱스를 반환
- `음수` : 찾고자 하는 값이 배열에 없는 경우
    - 계산식 : `-(insertion point) - 1`
    - `insertion point` : 만약 이 값이 배열에들어간다면 위치해야 할 인덱스

```java
import java.util.Arrays;

public class BinarySearchExample {
    public static void main(String[] args) {
        int[] scores = {50, 65, 70, 85, 90, 95}; 
        
        // 1. 존재하는 값을 찾을 때
        int target1 = 85;
        int index1 = Arrays.binarySearch(scores, target1);
        System.out.println(target1 + "의 인덱스: " + index1); // 출력: 3

        // 2. 존재하지 않는 값을 찾을 때
        int target2 = 80;
        int index2 = Arrays.binarySearch(scores, target2);
        
        // 80은 70(인덱스2)과 85(인덱스3) 사이에 있어야 하므로 삽입 지점은 3
        // 결과값: -(3) - 1 = -4
        System.out.println(target2 + "의 결과값: " + index2); // 출력: -4

    }
}
```

### 메소드
| 메소드 | 소속 클래스 | 용도 | 반환 타입 |
| :--- | :--- | :--- | :--- |
| `Arrays.binarySearch(arr, key)` | `java.util.Arrays` | 배열에서 `key`의 위치를 이진 탐색으로 찾음 | `int` |
| `Collections.binarySearch(list, key)` | `java.util.Collections` | 리스트에서 `key`의 위치를 이진 탐색으로 찾음 | `int` |
