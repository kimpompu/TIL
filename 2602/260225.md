# 260225 TIL

## Host
- 네트워크 환경에서 IP 주소를 가지고 네트워크에 연결된 모든 장치
- 종류
    - 로컬 호스트
    - 원격 호스트
    - 가상 호스트

## 포트
- `HTTP` : `80`
- `HTTPS` : `443`
- `MySQL` : `3306`
- `PostgreSQL` : `5432`
- `Spring Boot`(default): `8080`

## Host 설정
```yaml
server:
  port: 8080 

database:
  url: jdbc:mysql://db-server-hostname:3306/mydb 
```
