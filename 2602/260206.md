# 260206 TIL

## 유형별 로직
|문제 유형|	추천 로직|
|--------|-------------------------|
|자릿수 하나씩 검사하기|	`while (n > 0)` 문 안에서 `n % 10` 하기|
|최대/최소값 찾기|	`int max = 0;` 설정 후 비교하며 갱신하기|
|가장 가까운 값 찾기|	`Math.abs(기준 - 현재값)`으로 거리 비교하기|
|특정 조건 개수 세기|	`int count = 0;` 설정 후 조건 맞으면 `count++`|

## 오늘 사용한 메소드
| 메소드명 | 설명 | 사용 예시 |
| :--- | :--- | :--- |
 `Math.abs(n)` | 숫자의 **절댓값**을 반환합니다. | `Math.abs(-5)` → `5` |
 | `String.charAt(index)` | 문자열의 특정 **인덱스 문자를 반환**합니다. | `"hello".charAt(1)` → `'e'` |
| `String.valueOf(n)` | 다양한 타입을 **문자열로 변환**합니다. | `String.valueOf(10)` → `"10"` |
| `Character.isUpperCase(c)` | 문자가 **대문자인지 확인**하여 true/false 반환. | `Character.isUpperCase('A')` → `true` |
| `Character.toLowerCase(c)` | 문자를 **소문자로 변환**합니다. | `Character.toLowerCase('A')` → `'a'` |
