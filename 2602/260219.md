# 260219 TIL

## 코딩테스트 풀이에 사용한 메소드
| 분류 | 메서드 | 설명 | 예시 |
| :--- | :--- | :--- | :--- |
| **문자열 추출** | `substring(s, e)` | 시작(s)부터 끝(e-1)까지 문자열을 자름 | `"Hello".substring(0, 2)` → `"He"` |
| **진법 변환** | `Integer.toBinaryString(n)` | 정수를 2진수 형태의 문자열로 변환 | `Integer.toBinaryString(6)` → `"110"` |
| **대소문자 판별** | `Character.toUpperCase(c)` | 특정 문자를 대문자로 변환 | `Character.toUpperCase('a')` → `'A'` |
