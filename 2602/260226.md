## Integer bit 메소드
| 메소드 | 설명 | 예시  |
| :--- | :--- | :--- |
| `Integer.bitCount(n)` | `정수`를 `2진수`로 변환해 **1의 개수**를 반환 | `bitCount(10)` → `2` |
| `Integer.toBinaryString(n)` | `정수`를 **2진수 문자열**로 변환 | `toBinaryString(10)` → `"1010"` |
| `Integer.highestOneBit(n)` | `가장 왼쪽(큰 값)`에 있는 **1만 남기고 나머지를 0**으로 바꿈 | `highestOneBit(10)` → `8` (`1000`) |
| `Integer.lowestOneBit(n)` | `가장 오른쪽(작은 값)`에 있는 **1만 남기고 나머지를 0**으로 바꿈 | `lowestOneBit(10)` → `2` (`0010`) |

## 비트 연산자
| 연산자 | 이름 | 설명 | 예시 |
| :--- | :--- | :--- | :--- |
| `&` | AND | 둘 다 1일 때만 1 | `5 & 3` → `0001` (1) |
| `\|` | OR | 하나라도 1이면 1 | `5 \| 3` → `0111` (7) |
| `^` | XOR | 다를 때만 1 | `5 ^ 3` → `0110` (6) |
| `<<` | Left Shift | 왼쪽으로 밀기(`곱하기 2`) | `5 << 1` → `1010` (10) |
| `>>` | Right Shift | 오른쪽으로 밀기(`나누기 2`) | `5 >> 1` → `0010` (2) |
