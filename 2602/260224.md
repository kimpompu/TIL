# 260224 TIL

## 사용 메소드
| 메소드 | 설명 | 예시 |
| :--- | :--- | :--- |
| `Character.isDigit(c)` | 문자 `c`가 숫자인지 확인하여 `boolean`를 반환 | `Character.isDigit('1')` -> `true` |
| `Character.getNumericValue(c)` | 숫자 모양의 문자('5')를 정수(5)로 변환 | `getNumericValue('5')` -> `5` |
| `Math.pow(a, b)` | $a$를 $b$번 곱한 값($a^b$)을 계산(결과값은 실수형) | `Math.pow(3, 2)` -> `9.0` |
| `getOrDefault(k, d)` | 이름이 있으면 그 숫자를, 없으면 기본값(d)을 반환| `map.getOrDefault("new", 0)` |
| `keySet()` | map의 `key`들을 `Set`으로 반환 | `for (String name : map.keySet())` |
