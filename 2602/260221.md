## 노드

### 패킷
- 정의 : 노드가 네트워크를 통해 전송하는 데이터의 단위
- 구성 : 헤더 + 페이로드


## 네트워크
- 정의 : 노드들이 데이터를 주고받기 위해 연결된 시스템

## TCP/IP 계층

### 애플리케이션 계층
- 프로토콜 : `HTTP`, `FTP`, `SMTP`

### 전송 계층
- 프로토콜 : `TCP`, `UDP`

### TCP
- 연결 기반 프로토콜
- 두 노드간 연결(`3-Way Handshake`) 확인 후 데이터 전송
    - 3-Way Handshake : `SYN` - `SYN-ACK` - `SYN`
- 장점 : `신뢰성`

### UDP
- 연결과정 없이 데이터 전송
- 순서없음

### QUIC
- UDP 기반 
- TCP 연결 관리기능 제공
- `연결 ID`를 포함해 데이터 전송
- `TLS` 통합해 패킷 암호화하여 전송
    - TCP 기반 HTTPS에서 3-Way Handshake와 TIL Handshake를 진행하는데 이를 단축
- 멀티플렉싱 지원
