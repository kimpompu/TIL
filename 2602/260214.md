## 평행과 기울기
- 평행하다는 것은 `기울기가 같다`
- 두 점 $(x_1, y_1)$과 $(x_2, y_2)$를 지나는 직선의 기울기는 $m = \frac{y_2 - y_1}{x_2 - x_1}$
- `(double) (y2 - y1) / (x2 - x1)`

## 유한소수
- 기약분수로 나타냈을 때, 분모의 소인수가 2와 5만 존재
1. **분자와 분모의 최대공약수(GCD)** 계산
2. 분모를 소인수분해하여 2와 5 외의 숫자가 남는지 확인
    
    ```Java
    while (denominator % 2 == 0) {
            denominator /= 2;
        }
    while (denominator % 5 == 0) {
            denominator /= 5;
        }

    return (denominator == 1) ? true : false;        
    ```
