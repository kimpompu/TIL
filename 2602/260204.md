## 정규표현식
| 패턴 | 의미 | 설명 |
| :--- | :--- | :--- |
| `[abc]` | 집합 | a, b, c 중 문자 하나와 일치 |
| `[^abc]` | 부정 | a, b, c를 제외한 문자 하나와 일치 |
| `[a-z]` | 범위 | 소문자 a부터 z까지 중 하나 |
| `[0-9]` 또는 `\d` | 숫자 | 모든 숫자와 일치 |
| `^[a-zA-Z]` | 시작 | 영문자로 시작하는 패턴 |
| `.*` | 와일드카드 | 0개 이상의 모든 문자 |

## stream
| 함수 | 역할 | 사용 예시 |
| :--- | :--- | :--- |
| `stream()` | 스트림 생성 | `set.stream()` |
| `chars()` | 문자열을 정수 스트림으로 | `my_string.chars()` |
| `filter()` | 조건에 맞는 요소만 선택 | `.filter(Character::isDigit)` |
| `map()` | 가공 | 요소를 다른 형태로 변환 | `.map(s -> s.toUpperCase())` |
| `mapToInt()` | 정수형 스트림으로 변환 | `.mapToInt(Integer::intValue)` |
| `sorted()` | 오름차순 정렬 | `.sorted()` |
| `toArray()` | 배열로 최종 변환 | `.toArray()` |
| `distinct()` | 가공 | 중복된 요소를 제거 | `.distinct()` |
| `limit(n)` | 가공 | 앞에서부터 n개만 선택 | `.limit(3)` |
| `anyMatch()` | 검사 | 조건에 맞는게 하나라도 있는지 확인 | `.anyMatch(n -> n > 10)` |
| `sum()` | 결과 | 모든 요소의 합계 계산 | `.mapToInt(i -> i).sum()` |
| `collect()` | 결과 | List, Set 등 컬렉션으로 변환 | `.collect(Collectors.toList())` |

## 메소드 참조(`::`)
| 유형 | 문법 | 람다 표현식 예시 | 메서드 참조 예시 |
| :--- | :--- | :--- | :--- |
| **정적 메서드** | `클래스::메서드` | `s -> Integer.parseInt(s)` | `Integer::parseInt` |
| **인스턴스 메서드** | `객체::메서드` | `s -> System.out.println(s)` | `System.out::println` |
| **특정 타입 인스턴스 메서드** | `클래스::메서드` | `(s1, s2) -> s1.equals(s2)` | `String::equals` |
| **생성자** | `클래스::new` | `() -> new ArrayList<>()` | `ArrayList::new` |
