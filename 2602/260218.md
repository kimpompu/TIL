# Stack
| 메소드명 | 역할 | 비고 |
| :--- | :--- | :--- |
| `push(E item)` | 스택의 맨 위에 데이터를 추가 | 추가된 항목 리턴 |
| `pop()` | 스택의 맨 위 데이터를 꺼내며 삭제 | 스택이 비어있을 때 호출 시 `EmptyStackException` 발생 |
| `peek()` | 스택의 맨 위 데이터를 삭제하지 않고 읽기 | 현재 '직전 값'이 무엇인지 확인 |
| `isEmpty()` | 스택이 비어있는지 확인하여 `boolean`을 반환 | 첫 원소를 넣거나 `pop`을 하기 전 안전하게 확인하기 위해 사용 |
