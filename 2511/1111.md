## FeignClient에서 PATCH Method 사용 불가
### 원인
- FeignClient가 사용하는 `HttpURLConnection` 자바 객체가 PATCH METHOD 미지원
- 지원 METHOD
    - GET
    - POST
    - HEAD
    - OPTIONS
    - PUT
    - DELETE
    - TRACE

### 시도한 방법
1. `okHttpClient`
    - 의존성 추가
    - config 설정
    ```Java
    @Configuration
    class FeignOkHttpConfiguration {
        @Bean
        fun client(): OkHttpClient {
            return OkHttpClient()
        }
    }
    ```
    `실패`

### 실패 이유
- 서비스 이름을 IP로 해석하지 못하고, 일반 DNS 룩업을 시도하다가 실패
