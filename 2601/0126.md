## 배열, 리스트, 해시맵, 셋 비교

| 구분 | 배열(Array) | 리스트(List) | 해시맵(Map) | 셋(Set) |
| :--- | :--- | :--- | :--- | :--- |
| 선언 | int[] arr = {1, 2}; | List<Integer> list = new ArrayList<>(); | Map<K, V> map = new HashMap<>(); | Set<Integer> set = new HashSet<>(); |
| 크기 | arr.length | list.size() | map.size() | set.size() |
| 데이터 추가 | arr[i] = 10; | list.add(value); | map.put(key, value); | set.add(value); |
| 데이터 삭제 | 불가능 | list.remove(index/val); | map.remove(key); | set.remove(value); |
| 사이즈 | 고정 | 가변 | 가변 | 가변 |
| 중복 허용 | 허용 | 허용 | Key(불가), Value(허용) | 불가 |
| 순서 | 있음 | 있음 | 없음 | 없음 |

* 배열의 length는 필드라서 메소드 호출없이 바로 사용한다.
