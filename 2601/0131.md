### 알고리즘 문제
| 문제 | 키워드 | 공식 |
|:---|:---|:---|
| 배열 자르기 | Index 범위 | `Arrays.copyOfRange(arr, start, end + 1)` |
| 배열 자르기 | 배열 크기| `end - start + 1` |
| 숫자 알파벳 변환 | ASCII | `(char)(str.charAt(i) - '0' + 'a')` |
| 순서쌍 | 약수 | `n % i == 0` |

### Stream API 메서드 
| 메서드 | 역할 
| :--- | :--- |
| **`Arrays.stream()`** | 배열을 스트림으로 변환 | 
| **`IntStream.rangeClosed(start, end)`** | 특정 범위의 배열 생성 |
| **`.boxed()`** | 기본 타입(int)을 객체(Integer)로 변환 |
| **`.sorted()`** | 데이터 정렬<br>오름차순 : `.sorted()`<Br>내림차순 : `.sorted(Comparator.reverseOrder())` |
| **`.filter()`** | 조건에 맞는 데이터만 통과 |
| **`.map(e -> ...)`** | 데이터를 다른 형태로 변환 | 
| **`.count()`** | 요소의 총 개수를 반환 | 
| **`.toArray() / .collect()`** | 결과를 배열이나 리스트로 변환<br>`.collect(Collectors.toList())`| 

### indexOf
| 대상 | indexOf() 사용 가능 여부 | 비고 |
| :--- | :---: | :--- |
| **String** | 가능 | `str.indexOf("a")` |
| **List / ArrayList** | 가능 | `list.indexOf(value)` |
| **일반 배열 (Array)** | 불가능 | List 변환 후 사용하거나 for문 활용 |
