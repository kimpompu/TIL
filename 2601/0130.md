|----------|------------------|------------------------|

## 중복 문자 제거하기
```Java
String str = "banana";
StringBuilder sb = new StringBuilder();
for (int i = 0; i < str.length(); i++) {
    if (str.indexOf(str.charAt(i)) == i) {
        sb.append(str.charAt(i));
    }
}
String result = sb.toString(); 
```

### `indexOf(String)`
- 문자열에서 특정 문자 혹은 문자열이 처음으로 등장한 위치를 반환
- 없다면 -1 반환

```Java
import java.util.stream.Collectors;

String result = my_string.chars()              // 문자열을 정수 스트림으로 변환
    .distinct()                                // 중복 제거
    .mapToObj(c -> String.valueOf((char)c))    // 다시 문자로 변환
    .collect(Collectors.joining());            // 하나로 합치기
```

### chars()
- stream에서 문자열을 IntStream으로 변환
- `mapToObj(c -> String.valueOf((char) c))`로 다시 char 로 만듦

```Java
import java.util.LinkedHashSet;
import java.util.Set;

class Solution {
    public String solution(String my_string) {
        // 1. Set 선언
        Set<Character> set = new LinkedHashSet<>();
        
        // 2. 문자열을 한 글자씩 Set에 담기
        // Set은 중복된 값이 들어오면 자동으로 무시
        for (char c : my_string.toCharArray()) {
            set.add(c);
        }
        
        // 3. Set에 담긴 문자들을 다시 하나의 문자열로 만듦
        StringBuilder sb = new StringBuilder();
        for (Character c : set) {
            sb.append(c);
        }
        
        return sb.toString();
    }
}
```
