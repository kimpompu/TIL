## String
- 문자열을 뒤집거나 특정 문자를 제거
1. `String.split("")` : 문자열을 한 글자씩 잘라 배열로 만듦
2. `String.replace("A", "B")` : 특정 문자를 다른 문자로 치환
3. `StringBuilder` : 문자열을 효율적으로 뒤집을 때 사용(`sb.reverse().toString()`)
4. `String.toLowerCase()` / `toUpperCase()` : 대소문자 변환

## char
1. `(int)c -' '0'` : 문자 5를 숫자 5로 변환
2. `c + 32` : 대문자에서 소문자로 변환
3. `c - 32` : 소문자에서 대문자로 변환

## String ↔ char
1. `str.charAt(index)` : 특정 위치의 문자 하나를 반환
2. `str.toCharArray()` : 문자열을 char[] 배열로 변환
3. `String.valueOf(c)` : 문자를 문자열로 변환
4. `new String(charArray)` : 문자 배열을 문자열로 변환

## StringBuilder
| 메소드 | 설명 | 활용 예시 |
| :--- | :--- | :--- |
| **`.append(값)`** | 뒤에 문자열이나 문자를 추가 | `sb.append("abc").append('d')` |
| **`.insert(index, value)`** | 특정 인덱스에 값 삽입 | `sb.insert(1, "test")` |
| **`.delete(start, end)`** | 범위 내 문자 삭제(끝은 미포함) | `sb.delete(0, 2)` |
| **`.deleteCharAt(index)`** | 특정 위치의 문자 하나만 삭제 | `sb.deleteCharAt(3)` |
| **`.setCharAt(index, value)`** | 특정 위치의 문자를 변경 | `sb.setCharAt(0, 'A')` |
| **`.reverse()`** | 순서를 거꾸로 뒤집기 | `new StringBuilder("string").reverse()` |
| **`.toString()`** | StringBuilder를 String으로 변환 | `sb.toString()` |
