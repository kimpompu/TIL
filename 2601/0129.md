## Map

* Key-Value : 키(Key)는 중복될 수 없으며, 값(Value)은 중복될 수 있음
* 순서 : `HashMap`은 저장 순서를 보장하지 않음(순서가 중요하다면 `LinkedHashMap`이나 `TreeMap` 사용)

```JAVA
import java.util.HashMap;
import java.util.Map;

public class MapExample {
    public static void main(String[] args) {
        Map<String, Integer> map = new HashMap<>();

        // 1. 데이터 추가 및 수정
        map.put("Apple", 1000);
        map.put("Banana", 2000);
        map.put("Apple", 1500); // 키가 중복되면 기존 값이 덮어씌워짐

        // 2. 데이터 가져오기
        int price = map.get("Apple"); // 1500 반환 (없으면 null)

        // 3. 존재 여부 확인
        boolean hasApple = map.containsKey("Apple");   
        boolean hasValue = map.containsValue(2000);    

        // 4. 데이터 삭제
        map.remove("Banana");

        // 5. 데이터 개수 확인 및 비우기
        map.size();
        map.isEmpty();
        // map.clear();

        // 6. 키가 있으면 그 값을, 없으면 0(기본값)을 반환
        map.put("Grape", map.getOrDefault("Grape", 0) + 1);

        // 7. 키가 없을 때만 데이터를 추가
        map.putIfAbsent("Orange", 3000);

        // 전체 순회 방법 1: entrySet() 사용 (Key와 Value가 모두 필요할 때 - 가장 권장)
        for (Map.Entry<String, Integer> entry : map.entrySet()) {
            System.out.println("Key: " + entry.getKey() + ", Value: " + entry.getValue());
        }

        // 전체 순회 방법 2: keySet() 사용 (Key만 필요하거나 Key를 통해 Value를 찾을 때)
        for (String key : map.keySet()) {
            System.out.println("Key: " + key + ", Value: " + map.get(key));
        }

        // 전체 순회 방법 3: Lambda & forEach
        map.forEach((key, value) -> {
            System.out.println(key + " : " + value);
        });
    }
}
```
