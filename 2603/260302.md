## 원형탐색
1. `배열`, `List`
- `index % size`
```java
List<String> list = Arrays.asList("A", "B", "C");
int size = list.size();

for (int i = 0; i < 100; i++) { 
    System.out.println(list.get(i % size)); 
}
```
2. `Set`,`Map`
    1. `Iterator`, `향상된 for문`
    ```java
        Set<String> set = new HashSet<>(Arrays.asList("사과", "배", "포도"));

        while (true) { 
            for (String item : set) {
                System.out.println(item);
                // 특정 조건 시 break;
            }
        }
    ```
    2. `Stream` 사용
    ```java
    List<String> list = Arrays.asList("Red", "Green", "Blue");

    Stream.generate(() -> list)
        .flatMap(List::stream)
        .forEach(System.out::println);
    ```
