## SQL문

**모든 직원 정보 조회**

```sql
SELECT * FROM EMP;
```

**특정 부서(10번) 소속 직원 조회**

```sql
SELECT * FROM EMP WHERE DEPTNO = 10;
```

**급여 내림차순 정렬 조회**

```sql
SELECT * FROM EMP ORDER BY SAL DESC;
```

**이름에 'A'가 포함된 직원 조회**

```sql
SELECT * FROM EMP WHERE ENAME LIKE '%A%';
```

**급여가 2000 이상 3000 이하인 직원 조회**

```sql
SELECT * FROM EMP WHERE SAL BETWEEN 2000 AND 3000;
```

**중복을 제거한 부서 번호 조회**

```sql
SELECT DISTINCT DEPTNO FROM EMP;
```

**전체 직원 수 및 평균 급여 조회**

```sql
SELECT COUNT(*), AVG(SAL) FROM EMP;
```

**부서별 직원 수 조회**

```sql
SELECT DEPTNO, COUNT(*) 
FROM EMP 
GROUP BY DEPTNO;
```

**부서별 평균 급여가 2000 이상인 부서 조회**

```sql
SELECT DEPTNO, AVG(SAL) 
FROM EMP 
GROUP BY DEPTNO 
HAVING AVG(SAL) >= 2000;
```

**직원 이름과 소속 부서명 함께 조회** 

```sql
SELECT E.ENAME, D.DNAME 
FROM EMP E 
JOIN DEPT D ON E.DEPTNO = D.DEPTNO;
```

**부서가 없는 직원도 포함하여 조회**

```sql
SELECT E.ENAME, D.DNAME 
FROM EMP E 
LEFT JOIN DEPT D ON E.DEPTNO = D.DEPTNO;
```

**가장 급여가 높은 직원 조회**

```sql
SELECT ENAME, SAL 
FROM EMP 
WHERE SAL = (SELECT MAX(SAL) FROM EMP);
```

**새로운 직원 데이터 삽입**

```sql
INSERT INTO EMP (EMPNO, ENAME, SAL, DEPTNO) 
VALUES (1001, 'KIM', 3000, 10);
```

**특정 직원의 급여 수정**

```sql
UPDATE EMP 
SET SAL = 3500 
WHERE EMPNO = 1001;
```

**특정 직원 데이터 삭제**

```sql
DELETE FROM EMP WHERE EMPNO = 1001;
```

**급여 등급 매기기** 

```sql
SELECT ENAME, SAL,
    CASE 
        WHEN SAL >= 3000 THEN 'A등급'
        WHEN SAL >= 2000 THEN 'B등급'
        ELSE 'C등급'
    END AS GRADE
FROM EMP;
```

**테이블 생성**

```sql
CREATE TABLE EMP_COPY (
    EMPNO INT PRIMARY KEY,
    ENAME VARCHAR(50),
    SAL INT
);
```

**검색 속도 향상을 위한 인덱스 생성**
```sql
CREATE INDEX IDX_EMP_ENAME ON EMP(ENAME);
```
