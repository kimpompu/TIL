## 람다식 정렬
- `(a, b) -> (value)`에서 `value`에 따라 위치가 달라짐
    1. 음수 : `a`, `b` 순서
    2. 0 : `a`, `b` 순서
    3. 양수 : `b`, `a` 순서

### 오름차순
```java
// 1차원 배열
Arrays.sort(arr, (a, b) -> a - b); 

// 2차원 배열 
Arrays.sort(matrix, (a, b) -> a[0] - b[0]);
```

### 내림차순
```java
// 1차원 배열
Arrays.sort(arr, (a, b) -> b - a);

// 2차원 배열 
Arrays.sort(matrix, (a, b) -> b[1] - a[1]);
```

### 커스텀
```java
Arrays.sort(targets, (a, b) -> {
    if (a[0] == b[0]) {
        return b[1] - a[1]; // 시작점이 같으면 끝점 기준 내림차순
    }
    return a[0] - b[0];     // 기본은 시작점 기준 오름차순
});
```

| 정렬 종류 | 코드 형태 (람다) | 설명 |
| :--- | :--- | :--- |
| **오름차순** | `(a, b) -> a - b` | 작은 값 $\rightarrow$ 큰 값 |
| **내림차순** | `(a, b) -> b - a` | 큰 값 $\rightarrow$ 작은 값 |
| **절대값 기준** | `(a, b) -> Math.abs(a) - Math.abs(b)` | 원점에서 가까운 순서대로 |
| **문자열 길이** | `(a, b) -> a.length() - b.length()` | 짧은 단어부터 정렬 |
