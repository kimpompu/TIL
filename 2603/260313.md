## JavaScript와 TypeScript 비교
|구분|`JavaScript`|`TypeScript`|
|:---:|:---:|:---:|
|타입 결정|코드 실행(`Runtime`)|코드 작성 및 컴파일(`Compile-time`)|
|에러 발견|실행해봐야 버그를 알 수 있음|코드 작성 중에 빨간 줄로 알려줌|
|실행 환경|브라우저/Node.js에서 바로 실행|JS로 컴파일이 필요함|

## TypeScript
### 타입 선언
```TypeScript
let name: string = "Hello";
let age: number = 20;
let isTrue: boolean = true;
let person: { name: string; age: number } = {
  name: "jinyoung",
  age: 24
};
let arr: Array<number> = [10 , 20, 30];
```

### 함수 선언
```TypeScript
function add(a: number, b: number):number {
    return a+b;
}

add(10,20);
```


### Interface 선언
```TypeScript
interface User{
    id : number;
    name : string;
    email?: string; 
    // ?을 붙이면 필수가 아닌 선택사항
}

const master: User = {
    id:1,
    name:"Admin"
}
```
