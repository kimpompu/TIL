# 251208 TIL

## `multiSet(Map<K, V> map)`
- Redis 명령어: `MSET` 
- 동작 : Map에 들어있는 모든 Key-Value 쌍을 Redis에 저장합니다.
- 특징 : **원자성(Atomicity)**을 가짐. 즉, 모든 키가 저장되거나, 실패하면 아무것도 저장되지 않음

## `multiGet(List<String> keys)`
- Redis 명령어 : `MSET`
- 동작 : Map에 들어있는 모든 Key-Value 쌍을 Redis에 저장
- 특징 : **원자성(Atomicity)**을 가짐. 즉, 모든 키가 저장되거나, 실패하면 아무것도 저장되지 않음


| 메소드 (Java) | Redis 명령어 | 설명 | 시간 복잡도 |
| :--- | :--- | :--- | :--- |
| **`opsForValue()`** | - | String 타입 연산을 위한 객체 반환 | - |
| **`multiGet(keys)`** | **`MGET`** | 여러 Key의 값을 한 번에 조회 | O(N)  |
| **`multiSet(map)`** | **`MSET`** | 여러 Key-Value 쌍을 한 번에 저장 | O(N) |
| **`expire(key, ttl)`** | **`EXPIRE`** | 특정 Key의 만료 시간 설정 | O(1) |
