## CI/CD & GitHub Actions
- CI(Continuous Integration) : 코드를 합치고, 빌드하고, 테스트
- CD(Continuous Deployment) : 빌드된 결과물을 실제 서버에 배포
- Runner : GitHub Actions가 실행되는 가상의 컴퓨터(Ubuntu). 빌드를 마친 결과물(Docker Image)만 도커 허브에 올림

## Elastic IP(탄력적 IP)
- Public IP(공인 IP) : EC2를 껐다 켜면 바뀜(유동 IP)
- Elastic IP(탄력적 IP) : 사용자가 반납하기 전까지 바뀌지 않는 고정 IP

## workflow
```yml
name: Deploy Comment Service to EC2

on:
  push:
    branches: [ "main"]
    paths:
      - 'comment-service/**'

jobs:
  build-and-deploy:
    runs-on: ubuntu-latest

    steps:
      # 1. 코드 가져오기
      - name: Checkout code
        uses: actions/checkout@v4

      # 2. JDK 17 세팅
      - name: Set up JDK 17
        uses: actions/setup-java@v4
        with:
          java-version: '17'
          distribution: 'temurin'

      # 3. Gradle 실행 권한
      - name: Grant execute permission for gradlew
        working-directory: comment-service
        run: chmod +x gradlew

      # 4. Spring Boot 빌드
      - name: Build with Gradle
        working-directory: comment-service
        run: ./gradlew clean build -x test

      # 5. Docker Hub 로그인 
      - name: Login to Docker Hub
        uses: docker/login-action@v3
        with:
          username: ${{ secrets.DOCKER_USERNAME }}
          password: ${{ secrets.DOCKER_PASSWORD }}

      # 6. Docker 이미지 빌드 & 푸시
      - name: Build and Push Docker Image
        working-directory: comment-service
        run: |
          docker build -t ${{ secrets.DOCKER_USERNAME }}/comment-service:latest .
          docker push ${{ secrets.DOCKER_USERNAME }}/comment-service:latest

      # 7. EC2 배포
      - name: Deploy to EC2
        uses: appleboy/ssh-action@master
        with:
          host: ${{ secrets.EC2_HOST }}
          username: ubuntu
          key: ${{ secrets.EC2_SSH_KEY }}
          port: 22
          script: |
            cd ~/deploy
            
            sudo docker compose pull comment-service
            
            # 컨테이너 재시작
            sudo docker compose up -d comment-service
            
            # 청소
            sudo docker image prune -f
```
