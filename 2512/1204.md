# 251204 TIL

## MongoDB 선택
- 기존 postgreSQL에서 mongoDB로 변경하기로 결정

### 변경 이유
- 스키마 변경이 쉬움 : MongoDB는 스키마가 비고정
- 샤딩 지원
- TTL 인덱스 지원
    - 특정일이 지난 알림들 일괄 삭제 가능

| 특징 | 설명 |
| :--- | :--- |
| **기능 명칭** | TTL (Time To Live) Index |
| **핵심 역할** | 지정된 시간이 경과한 문서를 자동으로 영구 삭제 |
| **설정 옵션** | `expireAfterSeconds` (초 단위 설정) |
| **필수 조건** | 인덱스 필드가 **Date 타입**이어야 함 |
| **주요 용도** | 시스템 로그, 사용자 세션, 임시 인증 번호, SNS 알림, 장바구니 |
| **장점** | 삭제 배치(Batch) 작업을 별도로 구현할 필요가 없어 관리 비용 절감 |
