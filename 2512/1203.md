## JIT-PACK 배포
1. repository에 relese
2. JIP-PACK에서 getit

## 설정
1. build.gradel에서 삭제
```grdle
repositories {
	mavenCentral()
}
```
2. 의존성 추가
```grdle
	dependencies {
	        implementation 'com.github.****:태그'
	}
```

3. setting.gradle에 추가
```grdle
	dependencyResolutionManagement {
		repositoriesMode.set(RepositoriesMode.FAIL_ON_PROJECT_REPOS)
		repositories {
			mavenCentral()
			maven { url 'https://jitpack.io' }
		}
	}
```

4. 의존성을 공유하려면 implementation 대신 api로 기재
    - java.library로 써야함
