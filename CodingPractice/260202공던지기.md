# 공 던지기
```Java
class Solution {
    public int solution(int[] numbers, int k) {
        int answer = (2 * (k-1)) % numbers.length;
        return numbers[answer];
    }
}
```

```Java
class Solution {
    public int solution(int[] numbers, int k) {
        int answer = 0; 

        for (int i = 0; i < k - 1; i++) {
            answer = (answer + 2) % numbers.length;
        }

        return numbers[answer];
    }
}
```

## `원형`, `순환`, `반복`
1. 1번부터 시작해도 계산은 0번 인덱스에서 함(`k-1`)
2. 이동 공식
    * 현재 위치 : `current`
    * 이동 거리 : `step`
    * 목표 횟수 : `k`
    * 새로운 위치 : (`current` + `step`) % array.length
    * 해당 인덱스 : (`step` * (`k-1`)) % array.length
