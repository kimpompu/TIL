# n^2배열 자르기
```java
class Solution {
    public int[] solution(int n, long left, long right) {
        
        int size = (int) (right - left +1);
        int[] answer = new int[size];
        
        for(int i = 0; i < size; i++){
            long k = left + i;
            
            long row = k / n;
            long col = k % n;
            
            answer[i] = (int) Math.max(row,col) + 1;
        }
        
        return answer;
    }
}
```
## 1차원 배열의 인덱스 $k$를 $n$행 $n$열의 2차원 좌표로 변환
- 행(row): $k / n$
- 열(col): $k \% n$

## 좌표 $(row, col)$에서 값
- $Math.max(row, col) + 1$

### 값 계산
| 행\열 | 0열 | 1열 | 2열 |
| :---: | :---: | :---: | :---: |
| **0행** | (0,0) → **1** | (0,1) → **2** | (0,2) → **3** |
| **1행** | (1,0) → **2** | (1,1) → **2** | (1,2) → **3** |
| **2행** | (2,0) → **3** | (2,1) → **3** | (2,2) → **3** |
- $max(row, col)$ 로 인덱스 계산
- $max(row, col)$ + 1로 실제 값 계산
