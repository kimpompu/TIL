# 모스부호(1)
## 숫자를 알파벳으로 바꾸기
```Java
class Solution {
    public String solution(String letter) {
        
        String[] morse = {".-","-...","-.-.","-..",".","..-.","--.","....","..",".---","-.-",".-..","--","-.","---",".--.","--.-",".-.","...","-","..-","...-",".--","-..-","-.--","--.."};
        
        String[] words = letter.split(" ");
        StringBuilder sb = new StringBuilder();
        
        for(String word : words){
            for(int i = 0; i<morse.length;i++){
                if(word.equals(morse[i])){
                    sb.append((char)(i+'a'));
                    break;
                }
            }
        }
        
        String answer = sb.toString();
        return answer;
    }
}
```
