# 옷가게 할인 받기

```Java
class Solution {
    public int solution(int price) {
        double answer = (double) price;
        
        if(price >= 500000){
            answer = price * 0.8;
        }else if(price >= 300000){
            answer = price * 0.9;
        }else if(price >= 100000){
            answer = price * 0.95;
        }
        
        return (int)answer;
    }
}
```
