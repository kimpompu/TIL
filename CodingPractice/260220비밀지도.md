# 비밀지도

```Java
class Solution {
    public String[] solution(int n, int[] arr1, int[] arr2) {
        String[] answer = new String[n];
        
        for(int i=0; i<n;i++){
            String combined = Integer.toBinaryString(arr1[i] | arr2[i]);
            
            combined = String.format("%"+n+"s",combined);
            
            combined = combined.replaceAll("1","#");
            combined = combined.replaceAll("0"," ");
            
            answer[i] = combined;
        }
        
        return answer;
    }
}
```

## 사용 메소드
| 메소드/연산자 | 역할 |
| :--- | :--- |
| **`|`** | 두 정수를 이진수 기준으로 OR 연산 |
| **`Integer.toBinaryString()`** |  2진수 문자열로 바꿈 |
| **`String.format("%ns", str)`** |  문자열을 n글자 길이에 맞춰 오른쪽정렬하고 빈 칸을 채움 |
| **`String.format("%-ns", str)`** |  문자열을 n글자 길이에 맞춰 왼쪽정렬하고 빈 칸을 채움 |
