# 최댓값 만들기(2)

```Java
import java.util.Arrays;

class Solution {
    public int solution(int[] numbers) {
        
        Arrays.sort(numbers);
        
        int first = numbers[numbers.length - 2] * numbers[numbers.length - 1];
        int second = numbers[0] * numbers[1];
        
        int answer = first > second ? first: second;
        
        return answer;
    }
}
```
