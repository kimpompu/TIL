# 옹알이(1)

```Java
class Solution {
    public int solution(String[] babbling) {
        
        String[] words = {"aya", "ye", "woo", "ma"};
        int answer = 0;
        
        for(String s : babbling){
            for(String w : words){
                s = s.replace(w, " ");
            }
            
            if(s.trim().length()==0) answer++;
        }
        
        return answer;
    }
}
```
