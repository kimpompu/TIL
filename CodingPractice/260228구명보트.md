# 구명보트
```java
import java.util.*;

class Solution {
    public int solution(int[] people, int limit) {
        
        Arrays.sort(people);
        
        int answer = 0;
        int left = 0;
        int right = people.length - 1;
        
        while(left <= right){
            if(people[left] + people[right] <= limit){
                left++;
            }
            right--;
            answer++;
        }
        
        
        return answer;
    }
}
```

## `투 포인터`
- 배열을 정렬한 뒤, 양 끝단(가장 가벼운 사람과 가장 무거운 사람)을 가리키는 인덱스 변수를 사용

## 풀이법
1. 배열을 오름차순으로 정렬
2. `가장 가벼운 사람(left)`과 `가장 무거운 사람(right)`을 더함
3. 합 $\le$ limit : 두 명을 한 보트에 태우고 `left` 증가, `right` 감소
3-2. 합 $>$ limit : `무거운 사람`만 혼자 보트에 태우고 `right`만 감소
4. 모든 사람이 구조될 때까지 반복
