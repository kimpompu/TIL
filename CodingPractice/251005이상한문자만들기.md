# 이상한 문자 만들기

```Java
class Solution {
    public String solution(String s) {
        
        String answer = "";
        
        String[] arr = s.split("");
        int idx = 0;
        
        for(String str : arr){
            idx=str.contains(" ")?0:idx+1;
            
        // 짝수는 대문자, 홀수는 소문자
        // idx는 0부터 시작 -> 첫글자는 짝수
                    
            answer += idx%2==0?str.toLowerCase():str.toUpperCase();
        }
        

        
        return answer;
    }
}
```
