# 등수 매기기

```Java
class Solution {
    public int[] solution(int[][] score) {
        int length = score.length;
        int[] sums = new int[length];
        int[] answer = new int[length];
        
        for(int i=0; i<length; i++){
            sums[i] = score[i][0] + score[i][1];
        }
        
        for(int i=0; i<length; i++){
            int rank=1;
            for(int j=0; j<length; j++){
                if(sums[j] > sums[i]) rank++;
            }
            answer[i]=rank;
        }
        
        return answer;
    }
}
```
