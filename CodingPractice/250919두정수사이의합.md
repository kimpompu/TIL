# 두 정수 사이의 합

```Java
class Solution {
    public long solution(int a, int b) {
        long answer = 0;
        
        int big = a>b?a:b;
        int small = a>b?b:a;
        
        for(int i=small; i<=big; i++){
            if(i%1==0){
                answer += i;
            }
        }
        
        return answer;
    }
}
```
