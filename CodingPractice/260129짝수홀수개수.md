# 짝수 홀수 개수

```Java
class Solution {
    public int[] solution(int[] num_list) {
        int[] answer = new int[2];
        
        int odd = 0;
        int right = 0;
        
        for(int i=0; i<num_list.length; i++){
            if(num_list[i]%2==0){
                right++;
            }else{
                odd++;
            }
        }
        
        answer[0] = right;
        answer[1] = odd;
        
        return answer;
    }
}
```
