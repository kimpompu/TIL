# 로그인 성공?
```Java
class Solution {
    public String solution(String[] id_pw, String[][] db) {
        String id = id_pw[0];
        String pwd = id_pw[1];
        
        for(int i=0; i<db.length; i++){
            if(db[i][0].equals(id)){
                if(db[i][1].equals(pwd)){
                    return "login";
                }else{
                    return "wrong pw";
                }
            }
        }
        
        return "fail";
    }
}
```
