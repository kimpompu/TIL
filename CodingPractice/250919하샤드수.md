# 하샤드 수

```Java
class Solution {
    public boolean solution(int x) {
    boolean answer = true;
    
    //문자열로 변환
    String s = Integer.toString(x);
    char[] arr = s.toCharArray();
    
    int sum = 0;
    
    //자릿수의 합 구하기    
    for(int i = 0 ; i<arr.length ; i++){
        int j = arr[i] - '0';
        sum = sum + j;
    }
        
    answer = (x%sum==0) ? true : false ;
        
    return answer;
    }
}
```
