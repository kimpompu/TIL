# 숨어있는 숫자의 덧셈(2)

```Java
class Solution {
    public int solution(String my_string) {
        
        int answer = 0;
        
        String str = my_string.replaceAll("[a-zA-z]"," ");
        
        String[] arr = str.split(" ");
        
        for(String s : arr){
            if(!s.equals("")){
                answer += Integer.parseInt(s);
            }
        }

        return answer;
    }
}
```
