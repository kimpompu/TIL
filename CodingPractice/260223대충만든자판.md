# 대충 만든 자판
```java
import java.util.*;

class Solution {
    public int[] solution(String[] keymap, String[] targets) {
        
        int[] minClick = new int[26];
        Arrays.fill(minClick, Integer.MAX_VALUE);
        
        for(String key : keymap){
            for(int i = 0; i<key.length(); i++){
                int charIdx = key.charAt(i) - 'A';
                minClick[charIdx] = Math.min(minClick[charIdx], i+1);
            }
        }
        
        
        int[] answer = new int[targets.length];
        
        for(int i = 0; i<targets.length; i++){
            int total=0;
            boolean canMake = true;
            
            for(char c : targets[i].toCharArray()){
                int count = minClick[c-'A'];
                if(count==Integer.MAX_VALUE){
                    canMake = false;
                    break;
                }
                total += count;
            }
            
            answer[i] = canMake ? total : -1;
        }
        
        
        return answer;
    }
}
```

## 사용 메소드
| 메소드 | 설명 | 사용 예시 |
| :--- | :--- | :--- |
| `Arrays.fill(array, value)` | 배열의 모든 요소를 지정된 값으로 입력 | `Arrays.fill(minClick, MAX)` |
| `Integer.MAX_VALUE` | `int` 타입이 가질 수 있는 가장 큰 정수 값 반환 | |
