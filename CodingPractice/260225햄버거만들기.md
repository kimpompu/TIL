# 햄버거 만들기
```java
import java.util.Stack;

class Solution {
    public int solution(int[] ingredient) {
        int answer = 0;
        
        Stack<Integer> stack = new Stack<>();
        
        for(int item : ingredient){
            stack.push(item);
            
            if(stack.size() >=4 ){
                int size = stack.size();
                
                if(stack.get(size-4)==1 &&
                   stack.get(size-3)==2 &&
                   stack.get(size-2)==3 &&
                   stack.get(size-1)==1
                  ){
                    answer++;
                    
                    stack.pop();
                    stack.pop();
                    stack.pop();
                    stack.pop();                    
                }
            }
        }
        
        return answer;
    }
}
```

## Stack
| 메소드명 | 설명 | 예시 |
| :--- | :--- | :--- |
| `push(e)` | 스택의 맨 위에 데이터를 추가 | `stack.push(1);` |
| `pop()` | 스택의 맨 위에 있는 데이터를 꺼내서 제거 | `stack.pop();` |
| `peek()` | 맨 위의 데이터를 확인 | `stack.peek();` |
| `isEmpty()` | 스택이 비어있는지 확인 | `stack.isEmpty();` |
| `get(i)` | 특정 위치의 데이터를 확인 | `stack.get(size-1);` |
