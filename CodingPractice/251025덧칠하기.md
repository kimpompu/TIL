# 덧칠하기

```Java
import java.util.*;
class Solution {
    public int solution(int n, int m, int[] section) {
        int answer = 0;
        int painted = 0; // 마지막으로 칠한 구간의 끝 위치

        for (int s : section) {
            // 아직 칠하지 않은 구간이면 새로 롤러 사용
            if (s > painted) {
                answer++;
                painted = s + m - 1; // 이번에 칠한 구간의 끝
            }
        }

        return answer;
    }
}

```

> 그리디 알고리즘

- `최선의 선택`을 반복해서 전체적으로 최적의 결과를 얻는 방식
