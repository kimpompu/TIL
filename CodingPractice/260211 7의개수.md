# 7의 개수
```Java
class Solution {
    public int solution(int[] array) {

        int answer = 0;        

        for(int i : array){
            String str = String.valueOf(i);
            for(int j = 0; j<str.length(); j++){
                if(str.charAt(j)=='7') answer++;
            }
        }

        return answer;
    }
}
```
