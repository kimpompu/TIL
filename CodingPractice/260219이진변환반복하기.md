# 이진 변환 반복하기

```Java
class Solution {
    public int[] solution(String s) {
        
        int transCount = 0;
        int removeCount = 0;
        
        while(!s.equals("1")){
            int originLength = s.length();
            int oneCount = 0;
            
            for(int i = 0; i<s.length(); i++){
                if(s.charAt(i)=='1') oneCount++;
            }
            
            removeCount += (originLength - oneCount);
            
            s = Integer.toBinaryString(oneCount);
            
            transCount++;
        }
        
        
        int[] answer = {transCount, removeCount};
        return answer;
    }
}
```
