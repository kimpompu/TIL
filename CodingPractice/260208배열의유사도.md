# 배열의 유사도
```Java
import java.util.HashSet;
import java.util.Set;

class Solution {
    public int solution(String[] s1, String[] s2) {
        int answer = 0;
        
        Set<String> set = new HashSet<>();
        
        for(String str : s1){
            set.add(str);
        }
        
        for(String str : s2){
            if(set.contains(str)){
                answer++;
            }
        }
        
        return answer;
    }
}
```

| 메소드 | 설명 | 사용 예 |
|------|--------------|--------|
| add() | Set에 원소 추가 | set.add("a"); |
| contains() | 원소 포함 여부 확인 | set.contains("b"); |
| HashSet() | Set 생성 | new HashSet<>(); |
