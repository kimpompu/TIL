# 세균 증식

```Java
class Solution {
    public int solution(int n, int t) {
        int answer = n * (int)Math.round(Math.pow(2,t));
        return answer;
    }
}
```

```Java
class Solution {
    public int solution(int n, int t) {
        int answer = n << t;
        return answer;
    }
}
```

## <<
- `a << b`는 $a \times 2^b$와 같음
- ex) 2 << 10은 $2 \times 2^{10} = 2,048$
