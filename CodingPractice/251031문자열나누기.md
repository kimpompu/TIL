# 문자열 나누기

```Java
class Solution {
    public int solution(String s) {
        int answer = 0;      // 분리된 문자열의 개수
        int countX = 0;      // 기준 문자(x)의 개수
        int countOther = 0;  // x가 아닌 문자의 개수
        char x = ' ';        // 현재 기준 문자
        
        for (int i = 0; i < s.length(); i++) {
            // 첫 글자를 기준 문자로 설정
            if (countX == 0 && countOther == 0) {
                x = s.charAt(i);
            }
            
            // 기준 문자와 비교하여 각각의 개수를 증가
            if (s.charAt(i) == x) {
                countX++;
            } else {
                countOther++;
            }
            
            // 두 횟수가 같아지는 순간 분리
            if (countX == countOther) {
                answer++;
                countX = 0;
                countOther = 0;
            }
        }
        
        // 아직 남은 문자열이 있을 경우 마지막 조각 카운트
        if (countX != 0 || countOther != 0) {
            answer++;
        }
        
        return answer;
    }
}
```
