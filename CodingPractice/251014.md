# 가장 가까운 같은 글자

```Java
import java.util.*;
class Solution {
    public int[] solution(String s) {
        
        int[] answer = new int[s.length()];
        
        for(int i = 0; i<s.length(); i++){
            int idx = s.substring(0,i).lastIndexOf(s.charAt(i));
            if (idx != -1){
                answer[i] = i - idx;
            }else{
                answer[i] = idx;
            }
        }
        return answer;
    }
}
```
