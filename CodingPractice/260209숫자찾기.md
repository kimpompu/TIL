# 숫자 찾기

```Java
class Solution {
    public int solution(int num, int k) {
        String str = String.valueOf(num);
        String target = String.valueOf(k);
    
        int index = str.indexOf(target);
        
        int answer = (index == -1)? -1: index+1;
        
        return answer;
    }
}
```
