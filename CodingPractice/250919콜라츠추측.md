# 콜라츠 추측
```Java
class Solution {
    public int solution(int num) {
        
        long n = (long) num;
        
        int answer = 0;
              
        //num이 1이 될 때까지
        while(n!=1){
            
            //500번 수행시 -1
            if(answer>=500){
                return -1;
            }            

            //짝수라면 / 2
            if(n%2==0){
                n /= 2;
            }else{
            //홀수라면 * 3 + 1
                n = n * 3 + 1;
            }
                        
            answer++;           
        }

        return answer;
    }
}
```

> int num으로 처리했더니 입력값 `626331` 테스트에서 실패함. 검색해보니 int의 범위보다 큰 값이여서 `long`으로 변경해야 한다고 함.
