# 카펫
```java
class Solution {
    public int[] solution(int brown, int yellow) {
        
        int[] answer = new int[2];
        int total = brown + yellow;
        
        for(int h = 3; h <= Math.sqrt(total); h++){
            if(total % h == 0){
                int w = total / h;
                
                if((w-2) * (h-2) == yellow){
                    answer[0] = w;
                    answer[1] = h;
                    break;
                }
            }
        }
        
        
        return answer;
    }
}
```

## 풀이법
1. 전체 격자 수 total 계산(`brown + yellow`)
2. 세로 길이 `h`를 `최소값`인 `3`부터 `total의 제곱근`까지 1씩 증가시키며 반복
2-1. `최소값이 3인 이유` : `yellow`가 `1개` 이상 존재하려면 `가로와 세로가 최소 3 이상`이어야 함
3. `h`가 `total의 약수`인 경우, 가로 길이 `w`를 `total / h`로 계산
4. `(w - 2) * (h - 2) == yellow` 조건을 확인
5. 조건을 만족하면 `[w, h]`를 반환

## $\sqrt{N}$ 까지 탐색
- 두 약수 중 하나는 반드시 $\sqrt{N}$보다 작거나 같고, 다른 하나는 $\sqrt{N}$보다 크거나 같다
- $\sqrt{N}$까지만 확인하면 그 짝이 되는 큰 약수($w = total / h$)는 자동으로 계산

| 조건 유형 |예시 상황|응용 방법|
| :--- | :--- | :--- |
|`면적이 주어진 경우`|넓이가 `A`인 직사각형의 둘레 최소값 구하기|h를 1부터 $\sqrt{A}$까지 탐색하며 `w=A/h` 계산|
|`약수 개수/합 구하기`|자연수 `N`의 `모든 약수`를 구하라|i를 1부터 $\sqrt{N}$까지 반복하며 `i`와 `N/i`를 저장|
|`소수 판별`|특정 수 N이 소수인지 확인|`2`부터 $\sqrt{N}$까지만 나누어떨어지는지 확인|
|`격자 배치`|아이템 `N`개를 직사각형 형태로 배치할 때|h를 `1`부터 $\sqrt{N}$까지 탐색|
