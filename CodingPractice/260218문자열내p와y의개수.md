# 문자열 내 p와 y의 개수

```Java
class Solution {
    boolean solution(String s) {
        
        int count = 0;
        s = s.toLowerCase();
        
        for(int i=0; i<s.length(); i++){
            if(s.charAt(i) == 'p'){ 
                count++;
            }else if(s.charAt(i) == 'y'){
                count--;
            }
        }
        
        boolean answer = (count == 0);

        return answer;
    }
}
```
