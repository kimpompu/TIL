# 외계어 사전

```Java
import java.util.Arrays;

class Solution {
    public int solution(String[] spell, String[] dic) {
        // 1. 주어진 spell 배열을 정렬하고 하나의 문자열로 합침
        // 예: ["z", "d", "x"] -> "dxz"
        Arrays.sort(spell);
        String target = String.join("", spell);

        // 2. dic에 있는 단어들 하나씩 검사
        for (String word : dic) {
            // 3. 단어를 쪼개서 정렬
            char[] chars = word.toCharArray();
            Arrays.sort(chars);
            String sortedWord = new String(chars);

            // 4. 정렬된 결과가 target과 같다면 모든 알파벳을 한 번씩 사용
            if (sortedWord.equals(target)) {
                return 1;
            }
        }

        // 5. 미사용시 2를 반환
        return 2;
    }
}
```
