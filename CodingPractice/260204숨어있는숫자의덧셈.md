# 숨어있는 숫자의 덧셈(1)
```Java
class Solution {
    public int solution(String my_string) {
        
        String str = my_string.replaceAll("[^0-9]","");
        char[] numbers = str.toCharArray();
        int answer = 0;
        for(int i=0; i<numbers.length; i++){
            answer += (numbers[i]-'0');
        }
        
        
        return answer;
    }
}
```
