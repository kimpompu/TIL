# 컨트롤 제트

```Java
class Solution {
    public int solution(String s) {
        
        String[] arr = s.split(" ");        
        int answer = 0;
        int lastValue = 0;
        
        for(String str : arr){
            if(str.equals("Z")){
                answer -= lastValue;
            }else{
                lastValue = Integer.parseInt(str);
                answer += lastValue;
            }
        }
        
        return answer;
    }
}
```
