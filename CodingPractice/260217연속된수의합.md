# 연속된 수의 합
```Java
class Solution {
    public int[] solution(int num, int total) {
        int[] answer = new int[num];
        
        int sum=0;
        for(int i=0; i<num;i++){
            sum += i;
        }
        
        int start = (total-sum)/num;
        
        for(int i=0; i<num; i++){
            answer[i]=start+i;
        }
        
        return answer;
    }
}
```

## 공식
1. 0부터 num-1까지 다 더한 값을 구한다. ex) $num=3$이면 $0+1+2=3$
2. 전체 합($total$)에서 그 값을 뺀다
3. 남은 값을 $num$으로 나눈다
