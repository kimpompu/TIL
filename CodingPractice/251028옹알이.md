# 옹알이(2)

```Java
import java.util.regex.*;
import java.util.Arrays;

class Solution {
    public int solution(String[] babbling) {
        return (int) Arrays.stream(babbling)
            .filter(b -> !b.matches(".*(ayaaya|yeye|woowoo|mama).*")) // 같은 발음 연속 X
            .filter(b -> b.matches("^(aya|ye|woo|ma)+$")) // 네 발음으로만 구성
            .count();
    }
}
```
