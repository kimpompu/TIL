# 다음에 올 숫자
```Java
class Solution {
    public int solution(int[] common) {
        
        int length = common.length;
        int last = common[length-1];
        
        if(common[1]-common[0] == common[2]-common[1]){
            //등차
            int diff = common[1]-common[0];
            
            return last + diff;
            
        }else{
            //등비
            
            int ratio = common[1] / common[0];
            
            return last * ratio;
        }
        
    }
}
```
## 등차
- `공차` : `2번째 - 1번째`
## 등비
- `공비` : `2번째 / 1번째`
