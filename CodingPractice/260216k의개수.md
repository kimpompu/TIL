# k의 개수
```Java
class Solution {
    public int solution(int i, int j, int k) {
        int answer = 0;
        String target = String.valueOf(k);
        
        for(int n = i; n<=j; n++){
            String str = String.valueOf(n);
            
            for(int l=0; l<str.length(); l++){
                if(str.substring(l, l+1).equals(target)){
                    answer++;
                }
            }
        }
        
        return answer;
    }
}
```
