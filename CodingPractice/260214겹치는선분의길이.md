# 겹치는 선분의 길이

```Java
class Solution {
    public int solution(int[][] lines) {
        // 좌표 범위가 -100 ~ 100이므로 크기가 200인 배열 생성 (각 칸을 의미)
        int[] road = new int[200];
        
        for (int[] line : lines) {
            int start = line[0] + 100; // 음수 좌표를 양수 인덱스로 변환
            int end = line[1] + 100;
            
            // 시작점부터 끝점 '전'까지의 칸을 카운트함
            for (int i = start; i < end; i++) {
                road[i]++;
            }
        }
        
        int answer = 0;
        for (int count : road) {
            // 2개 이상의 선분이 지나간 칸의 개수를 합산
            if (count >= 2) {
                answer++;
            }
        }
        
        return answer;
    }
}
```

| 요소 |  설명 |
| :--- |  :--- |
| `int[] road` | 각 좌표 구간의 선분 중복 횟수를 기록 |
| `Offset (+100)` | 음수 범위를 배열 인덱스에 매핑하기 위해 수치 조정 |
