# 중복된 문자 제거

```Java
class Solution {
    public String solution(String my_string) {
        StringBuilder sb = new StringBuilder();
        String[] arr = my_string.split("");
        
        for(String s : arr){
            if(sb.indexOf(s)==-1){
                sb.append(s);
            }
        }
        
        String answer = sb.toString();
        return answer;
    }
}
```

| 클래스 | contains() 존재 여부 | 대체 메소드 |
| :--- | :---: | :--- |
| **String** | **O** | - |
| **List / Set** | **O** | - |
| **StringBuilder** | **X** | `indexOf()` |
| **Array (배열)** | **X** | `Arrays.asList(arr).contains()` 또는 반복문 |
