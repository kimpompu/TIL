# 가까운 수
```Java
import java.util.Arrays;

class Solution {
    public int solution(int[] array, int n) {
        Arrays.sort(array);
        int answer = array[0];
        for(int i=0; i<array.length; i++){
            int currentDiff = Math.abs(array[i]-n);
            int minDiff = Math.abs(answer-n);
            if(currentDiff<minDiff){
                answer=array[i];
            }
        }
        return answer;
    }
}
```
