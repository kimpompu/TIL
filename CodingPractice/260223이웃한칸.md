# 이웃한 칸

```java
class Solution {
    public int solution(String[][] board, int h, int w) {
        
        int n = board.length;
        
        int answer = 0;

        // 우,하,상,좌
        int[] dh = {0,1,-1,0};
        int[] dw = {1,0,0,-1};
        
        for(int i = 0; i< 4; i++){
            int h_check = h + dh[i];
            int w_check = w + dw[i];
            
            if(h_check >=0 && h_check < n && w_check >= 0 && w_check<n){
                if(board[h][w].equals(board[h_check][w_check])) answer++;
            }
        }
        

        return answer;
    }
}
```

## 2차원 배열 좌표
- 2차원 배열 **왼쪽 상단**이 $(0,0)$
    ```
    [0,0] [0,1] [0,2]
    [1,0] [1,1] [1,2] 
    [2,0] [2,1] [2,2]
    ```
