# 다트 게임
```java
class Solution {
    public int solution(String dartResult) {
        int[] scores = new int[3];
        int idx = -1;
        
        for(int i=0;i<dartResult.length();i++){
            char c = dartResult.charAt(i);
            
            if(Character.isDigit(c)){
                idx++;
                if(c=='1' && dartResult.charAt(i+1)=='0'){
                    scores[idx] = 10;
                    i++;
                }else{
                    scores[idx] = Character.getNumericValue(c);
                }
            }else if(c=='S'){
                scores[idx] = (int)Math.pow(scores[idx],1);
            }else if(c=='D'){
                scores[idx] = (int)Math.pow(scores[idx],2);
            }else if(c=='T'){
                scores[idx] = (int)Math.pow(scores[idx],3);
            }else if(c=='*'){
                scores[idx] *= 2;
                if(idx>0){
                    scores[idx-1] *=2;
                }
            }else if(c=='#'){
                scores[idx] *= -1;
            }
            
        }
        
        int answer = scores[0] + scores[1] + scores[2];
        
        return answer;
    }
}
```
## 메소드
| 메소드 | 설명 | 예시 |
| :--- | :--- | :--- |
| `Character.isDigit(c)` | 문자 `c`가 숫자인지 확인하여 `boolean`를 반환 | `Character.isDigit('1')` -> `true` |
| `Character.getNumericValue(c)` | 숫자 모양의 문자('5')를 정수(5)로 변환 | `getNumericValue('5')` -> `5` |
| `Math.pow(a, b)` | $a$를 $b$번 곱한 값($a^b$)을 계산(결과값은 실수형) | `Math.pow(3, 2)` -> `9.0` |
