# 푸드 파이터 대회

```Java
class Solution {
    public String solution(int[] food) {
        
        StringBuilder sb = new StringBuilder();
        
        for(int i = 1; i<food.length; i++){
            int num = food[i]/2;
            for(int j = 0; j<num ; j++){
                sb.append(i);
            }
        }
        
        String s = sb.toString();
        
        String answer = s+"0"+sb.reverse();
        return answer;
    }
}
```
