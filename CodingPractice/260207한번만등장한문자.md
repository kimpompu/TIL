# 한 번만 등장한 문자
```Java
class Solution {
    public String solution(String s) {
        
        int[] count = new int[26];
        
        for(char c: s.toCharArray()){
            count[c-'a']++;
        }
        
        StringBuilder sb = new StringBuilder();
        
        for(int i=0; i<26;i++){
            if(count[i]==1){
                sb.append((char)(i+'a'));
            }
        }
        
        String answer = sb.toString();
        return answer;
    }
}
```
