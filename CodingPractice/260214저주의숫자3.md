# 저주의 숫자 3

```Java
class Solution {
    public int solution(int n) {
        int answer = 0;
        
        for(int i = 1; i<=n; i++){
            answer++;
            
            while(answer%3==0 || String.valueOf(answer).contains("3")){
                answer++;
            }
            
        }
        
        return answer;
    }
}
```
