# 전화번호 목록
```java
import java.util.HashMap;
import java.util.Map;

class Solution {
    public boolean solution(String[] phone_book) {
        // 1. 모든 전화번호를 HashMap에 추가
        Map<String, Integer> map = new HashMap<>();
        for (int i = 0; i < phone_book.length; i++) {
            map.put(phone_book[i], i);
        }

        // 2. 모든 전화번호를 돌며 각 번호의 부분 문자열이 맵에 있는지 확인
        for (int i = 0; i < phone_book.length; i++) {
            // 해당 번호의 길이를 1부터 (자기 자신 길이 - 1)까지 슬라이스
            for (int j = 1; j < phone_book[i].length(); j++) {
                // 문자열을 앞에서부터 j번째까지 추출
                String prefix = phone_book[i].substring(0, j);
                
                // 추출한 문자열이 맵에 Key로 존재한다면 
                if (map.containsKey(prefix)) {
                    // 번호가 다른 번호의 접두어
                    return false;
                }
            }
        }

        return true;
    }
}
```

## 메소드
| 메소드 | 설명 | 시간 복잡도 |
| :--- | :--- | :--- |
| `map.put(K key, V value)` | 해시 맵에 키와 값을 저장 | $O(1)$ |
| `map.containsKey(Object key)` | 특정 키가 맵에 존재하는지 확인 | $O(1)$ |
