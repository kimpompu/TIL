# 소인수분해

```Java
import java.util.*;

class Solution {
    public int[] solution(int n) {
        
        Set<Integer> set = new LinkedHashSet<>();
        
        for(int i=2; i<=n; i++){
            while(n % i == 0){
                set.add(i);
                n /= i;
            }
        }
        
        
        return set.stream().mapToInt(Integer::intValue).toArray();
    }
}
```

## stream()

| 함수 | 역할 | 사용 예시 |
| :--- | :--- | :--- |
| `stream()` | 스트림 생성 | `set.stream()` |
| `mapToInt()` | 정수형 스트림으로 변환 | `.mapToInt(Integer::intValue)` |
| `toArray()` | 배열로 최종 변환 | `.toArray()` |
