# H-Index
```java
import java.util.*;

class Solution {
    public int solution(int[] citations) {
        
        int answer = 0;
        
        Arrays.sort(citations);
        int length = citations.length;
        
        for(int i = 0; i < length; i++){
            
            int candidate = length - i;
            
            if(citations[i] >= candidate){
                answer = candidate;
                break;
            }
            
        }
        
        return answer;
    }
}
```

## H-Index
- H-Index는 전체 논문 수 $n$을 넘을 수 없다.
- 정렬된 상태에서 $i$번째 논문의 인용 횟수가 $h$라고 할 때, 그 논문을 포함해 뒤에 남은 논문의 개수($n - i$)가 곧 $h$번 이상 인용된 논문의 개수
