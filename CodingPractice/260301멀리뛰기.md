# 멀리뛰기
```java
class Solution {
    public long solution(int n) {
        
        if(n == 1) return 1;
        
        long[] dp = new long[n+1];
        
        dp[1] = 1;
        dp[2] = 2;
        
        for(int i = 3; i <= n; i++){
            dp[i] = (dp[i-2] + dp[i-1]) % 1234567;
        }
        
        return dp[n];
    }
}
```

## 피보나치 수열
- $a_n = a_{n-1} + a_{n-2}$
