# 체육복

```Java
import java.util.*;

public class Solution {
    public int solution(int n, int[] lost, int[] reserve) {
        // 1️⃣ ArrayList로 변환하여 중복 처리 쉽게
        List<Integer> lostList = new ArrayList<>();
        List<Integer> reserveList = new ArrayList<>();

        for (int l : lost) lostList.add(l);
        for (int r : reserve) reserveList.add(r);

        // 2️⃣ 여벌이 있지만 도난당한 학생 제거 (자기 자신이 입어야 함)
        List<Integer> intersection = new ArrayList<>(lostList);
        intersection.retainAll(reserveList); // 교집합 구하기

        lostList.removeAll(intersection);
        reserveList.removeAll(intersection);

        // 3️⃣ 여벌을 빌려주기
        for (int i = 0; i < reserveList.size(); i++) {
            int reserveStudent = reserveList.get(i);

            // 바로 앞번호 학생이 잃어버렸으면 빌려주기
            if (lostList.contains(reserveStudent - 1)) {
                lostList.remove(Integer.valueOf(reserveStudent - 1));
            }
            // 앞번호가 없으면 뒷번호에게 빌려주기
            else if (lostList.contains(reserveStudent + 1)) {
                lostList.remove(Integer.valueOf(reserveStudent + 1));
            }
        }

        // 전체 학생 수 - 아직도 체육복이 없는 학생 수
        return n - lostList.size();
    }
}

```

> 사용 메소드
- retainAll(Collection<?> c) : 두 리스트의 교집합만 남김

- removeAll(Collection<?> c) : 리스트에서 다른 리스트의 모든 원소 제거
