# 다음 큰 숫자
```java
class Solution {
    public int solution(int n) {
        
        int target = Integer.bitCount(n);
        
        int next = n+1;
        
        while(true){
            
            if(Integer.bitCount(next) == target) break;
            
            next++;
        }
        
        return next;
    }
}
```

## 메소드
| 메소드 | 설명 | 예시 |
| :--- | :--- | :--- |
| `Integer.bitCount(int i)` | 정수를 2진수로 변환했을 때 **1의 개수**를 반환, 없으면 `0` | `bitCount(7)` (2진수 111) $\rightarrow$ 3 |
