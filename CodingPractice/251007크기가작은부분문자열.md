# 크기가 작은 부분 문자열

```Java
class Solution {
    public int solution(String t, String p) {
        int answer = 0;
        
        int len = p.length();
        Long lP = Long.parseLong(p); // 최대 18자리
        
        for(int i = 0; i< t.length() - len + 1; i++){
            String s = t.substring(i, i+len);
            
            if (Long.parseLong(s) <= lP){
                answer++;
            }
        }
        
        
        return answer;
    }
}
```
