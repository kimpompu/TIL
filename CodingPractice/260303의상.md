# 의상
```java
import java.util.*;

class Solution {
    public int solution(String[][] clothes) {
        
        HashMap<String, Integer> map = new HashMap<>();
        
        for(String[] cloth : clothes){
            String type = cloth[1];
            map.put(type, map.getOrDefault(type, 0)+1);
        }
        
        int answer = 1;
        
        for(int count : map.values()){
            answer *= (count + 1);
        }
        
        return answer - 1;
    }
}
```

## 조합
- $(count_1 + 1) \times (count_2 + 1) \times \dots \times (count_n + 1) - 1$

## 경우의 수
- `특정 종류 경우의 수` : 특정 종류의 의상이 $n$개 있다면, 선택지는 $n+1$
- `전체 조합` : 모든 종류 * $(n+1)$
- `예외 처리` : "최소 한 개의 의상은 입는다"라고 했으므로, **모든 종류를 하나도 입지 않은 경우**를 뺀다.
