# 소수 찾기
```java
class Solution {
    public int solution(int n) {
        int answer = 0;
        // n까지의 숫자를 체크하기 위해 n+1 크기 배열 생성
        // true면 소수, false면 소수 아님
        boolean[] isPrime = new boolean[n + 1];
        
        // 처음엔 모두 소수(true)로 초기화 (0과 1은 제외)
        for (int i = 2; i <= n; i++) {
            isPrime[i] = true;
        }

        // 에라토스테네스의 체 알고리즘
        for (int i = 2; i <= Math.sqrt(n); i++) {
            if (isPrime[i]) { // i가 소수라면
                // i의 배수들을 모두 false로 변경 (i*i부터 시작해도 됨)
                for (int j = i * i; j <= n; j += i) {
                    isPrime[j] = false;
                }
            }
        }

        // true로 남아있는 개수 세기
        for (int i = 2; i <= n; i++) {
            if (isPrime[i]) answer++;
        }

        return answer;
    }
}
```

## 소수찾기 풀이법
1. 배열 준비: 2부터 $n$까지의 숫자가 들어있는 리스트 선언(처음엔 모두 소수라고 가정)
2. 가장 작은 소수 2부터 2의 배수 지우기
3. 이 과정을 $\sqrt{n}$까지만 반복

# 사용 메소드
| 메소드 | 설명 | 예시 |
| :--- | :--- | :--- |
| **`Math.sqrt(n)`** | 숫자 $n$의 제곱근($\sqrt{n}$) 반환 | `Math.sqrt(25)` → 5.0 |
| **`Math.pow(n,m)`** | 숫자 $n$의 제곱($n^{m}$) 반환 | `Math.pow(2,3)` → 8.0 |
