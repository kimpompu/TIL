# 폰켓몬
```java
import java.util.HashSet;

class Solution {
    public int solution(int[] nums) {
        
        int max = nums.length / 2;
        
        HashSet<Integer> set = new HashSet<>();
        
        for(int n : nums){
            set.add(n);
        }
        
        int sort = set.size();
        
        if(sort > max){
            return max;
        }else{
            return sort;
        }
        
    }
}
```
