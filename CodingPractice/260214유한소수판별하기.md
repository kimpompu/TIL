# 유한소수 판별하기

```Java
class Solution {
    public int solution(int a, int b) {
        // 1. 최대공약수를 구해 기약분수의 분모를 만든다.
        int gcd = getGCD(a, b);
        int denominator = b / gcd;
        
        // 2. 분모에서 2를 모두 제거한다.
        while (denominator % 2 == 0) {
            denominator /= 2;
        }
        
        // 3. 분모에서 5를 모두 제거한다.
        while (denominator % 5 == 0) {
            denominator /= 5;
        }
        
        // 4. 남은 값이 1이면 유한소수(1), 아니면 무한소수(2)
        return (denominator == 1) ? 1 : 2;
    }
    
    // 유클리드 호제법을 이용한 GCD 계산
    private int getGCD(int a, int b) {
        if (b == 0) return a;
        return getGCD(b, a % b);
    }
}
```
