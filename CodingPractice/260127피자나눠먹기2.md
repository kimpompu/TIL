# 피자 나눠 먹기(2)

```Java
class Solution {
    public int solution(int n) {
        // 1. 6과 n의 최대공약수(GCD)를 구한다.
        int gcd = getGCD(6, n);
        
        // 2. 최소공배수(LCM)를 구한다. 
        // 공식: (a * b) / GCD
        int lcm = (6 * n) / gcd;
        
        // 3. 필요한 피자 판 수를 구한다. (전체 조각 / 6)
        return lcm / 6;
    }

    // 유클리드 호제법 함수
    private int getGCD(int a, int b) {
        if (b == 0) {
            return a;
        }
        return getGCD(b, a % b);
    }
}
```
