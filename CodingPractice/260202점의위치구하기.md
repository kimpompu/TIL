# 점의 위치 구하기 
```Java
class Solution {
    public int solution(int[] dot) {
        int answer = 0;
        
        boolean x = dot[0] > 0 ? true : false;
        boolean y = dot[1] > 0 ? true : false;
        
        if(x&&y){
            answer = 1;
        }else if(!x&&y){
            answer = 2;
        }else if(!x&&!y){
            answer = 3;
        }else if(x&&!y){
            answer = 4;
        }
        
        return answer;
    }
}
```
