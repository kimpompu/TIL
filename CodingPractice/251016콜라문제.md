# 콜라 문제
```Java
class Solution {
    public int solution(int a, int b, int n) {
        int answer = 0; // 총 받은 콜라 수
        
        while(n >= a) { // 최소 교환 가능한 빈 병 수가 있어야 반복 가능
            answer += (n / a) * b; // 교환해서 받은 콜라 수를 누적
            n = (n / a) * b + (n % a); 
            // 새로 받은 콜라(=빈병) + 교환 후 남은 빈 병 = 다음 반복의 빈 병 수
        }
        
        return answer;
    }
}
```
