# 실패율
```java
import java.util.*;

class Solution {
    public int[] solution(int N, int[] stages) {
        
        int[] stageCount = new int[N+2];
        
        for(int s : stages){
            stageCount[s]++;
        }
        
        List<Stage> stageList = new ArrayList<>();
        double totalPlayers = stages.length;
        
        for(int i = 1; i<=N; i++){
            if(totalPlayers == 0){
                stageList.add(new Stage(i, 0.0));
            }else{
                double failureRate = stageCount[i] / totalPlayers;
                stageList.add(new Stage(i, failureRate));
                totalPlayers -= stageCount[i];
            }
        }
        
        Collections.sort(stageList, (a,b) -> {
            if(a.rate == b.rate){
                return Integer.compare(a.id, b.id);
            }
            return Double.compare(b.rate, a.rate);
        });
        
        
        int[] answer = new int[N];
        for(int i = 0; i<N; i++){
            answer[i] = stageList.get(i).id;
        }
        
        return answer;
    }
    
    class Stage {
        int id;
        double rate;

        Stage(int id, double rate) {
            this.id = id;
            this.rate = rate;
        }
    }
}
```

## 메소드
| 메소드 | 설명 |
| :--- | :--- |
| `Collections.sort()` | 리스트에 들어있는 데이터를 특정 기준에 따라 정렬. 양수면 순서 바꿈 |
| `Integer.compare(a, b)` | 두 정수 a와 b를 비교 |
| `Double.compare(a, b)` | 두 실수(소수) a와 b를 비교 |


## compare(A, B)
- 결과가 `음수` : A가 B보다 작다 (`A < B`)
- 결과가 `0`: A와 B가 같다 (`A = B`)
- 결과가 `양수`: A가 B보다 크다 (`A > B`)
