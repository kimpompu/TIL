## replaceAll()
- public String replaceAll(String regex, String replacement)

### 정규표현식(regex)
| 패턴 | 의미 | 설명 |
| :--- | :--- | :--- |
| `[abc]` | 집합 | a, b, c 중 문자 하나와 일치 |
| `[^abc]` | 부정 | a, b, c를 제외한 문자 하나와 일치 |
| `[a-z]` | 범위 | 소문자 a부터 z까지 중 하나 |
| `[0-9]` 또는 `\d` | 숫자 | 모든 숫자와 일치 |
| `^[a-zA-Z]` | 시작 | 영문자로 시작하는 패턴 |
| `.*` | 와일드카드 | 0개 이상의 모든 문자 |

```Java
class Solution {
    public String solution(String my_string) {
        StringBuilder sb = new StringBuilder();
        
        for (int i = 0; i < my_string.length(); i++) {
            char c = my_string.charAt(i);
            
            if (c != 'a' && c != 'e' && c != 'i' && c != 'o' && c != 'u') {
                sb.append(c);
            }
        }
        
        return sb.toString();
    }
}
```
