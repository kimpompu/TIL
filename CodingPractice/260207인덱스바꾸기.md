# 인덱스 바꾸기
```Java
class Solution {
    public String solution(String my_string, int num1, int num2) {
        char[] ch = my_string.toCharArray();
        
        char temp = ch[num1];
        ch[num1] = ch[num2];
        ch[num2] = temp;
        
        return String.valueOf(ch);
    }
}
```

```Java
class Solution {
    public String solution(String my_string, int num1, int num2) {
        StringBuilder sb = new StringBuilder(my_string);
        
        char char1 = sb.charAt(num1);
        char char2 = sb.charAt(num2);
        
        sb.setCharAt(num1, char2);
        sb.setCharAt(num2, char1);
        
        return sb.toString();
    }
}
```

| 메소드명 | 설명 | 사용 예시 |
| :--- | :--- | :--- |
| `sb.charAt(index)` | 특정 인덱스에 있는 문자 반환 | `sb.charAt(1)` -> `'e'` |
| `sb.setCharAt(index, char)` | `StringBuilder` 내의 특정 인덱스 문자를 변경 | `sb.setCharAt(0, 'z')` |
