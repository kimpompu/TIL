# 정수 제곱근 판별   

> 처음 시도했던  코드 (실패)
```Java
class Solution {
    public long solution(long n) {
        long answer = 0;
        
        //i*i==n ? i+1 ; -1
        for(long i=1; i<n; i++){
            if(i*i==n){
                return answer=(i+1)*(i+1);
            }
            answer=-1;
        }
        
        return answer;
    }
}
```
* 테스트 18개 중 17개는 성공했는데 마지막 18번을 실패했다. 이유가 뭘까    
    * `i<n`을 `i<=n`으로 수정했더니 성공


> 검색 후 수정 코드(성공)
```Java
class Solution {
    public long solution(long n) {
        long answer = 0;
        
        //제곱근 구하기
        long sqrt = (long)Math.sqrt(n);
        
        //n이 맞다면
        if(sqrt * sqrt == n){
            //제곱근+1의 거듭제곱값 구하기
            return (long) Math.pow(sqrt+1,2);
        }

        //제곱근이 없다면 -1 반환
        answer=-1;
        
        return answer;
    }
}
```

> Math.sqrt(n) : double형의 제곱근을 반환   
> Math.pow(제곱근,제곱할횟수) : double형의 제곱값을 반환
