### 1. UUID(Universally Unique Identifier)
> ___정의___
128-bit 범용 고유 식별자

>___사용목적___
중복되지 않는 유일한 값 생성

>___구조___
![UUID 구조](https://www.techtarget.com/rms/onlineimages/time_based_uuid_example-f.png)
>출처 [What is a UUID (Universally Unique Identifier)?
](https://www.techtarget.com/searchapparchitecture/definition/UUID-Universal-Unique-Identifier)

> ___import___
```Java
 import java.util.UUID;

     UUID uuid = UUID.randomUUID();// 버전4 UUID값 호출

```

> PK 설정
```Java
@Id
@GeneratedValue(generator = "uuid2")
@GenericGenerator(name="uuid2", strategy = "uuid2")
@Column(name = "user_id", columnDefinition = "BINARY(16)")
private UUID id;
```

> PK 사용시 고려사항
 - 보안 : UUID PK를 사용하면 UUID 통해 다른 정보를 유추할 수 없음
 - DB성능
