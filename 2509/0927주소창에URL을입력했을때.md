# 1. 주소창에 URL을 입력했을 때 일어나는 일
1. 브라우저 주소창에 도메인(www.google.com) 입력
    1-1. 캐시(로컬/운영체제/라우터/ISP DNS 서버)에서 도메인 정보 조회
2. DNS 에서 해당 도메인을 조회해 IP 주소로 변환
3. 클라이언트 브라우저가 서버와 TCP 연결
4. 서버에 HTTP 요청
5. 웹 서버가 요청에 대한 응답 리소스(HTML, CSS 등)을 전송
6. 응답을 브라우저에서 랜더링
